# wheel-jquery-pulg-in
这是一个jquery轻量级的插件，用法与一些CSS框架类似，都是添加特定的类名，具体实现可到特定的文件夹中查看，而文件`wheel.js`就是总压缩js，其主要可以实现以下几个功能：

- [轮播](http://www.seieiblog.top/settimeplay)
- [模态框](http://www.seieiblog.top/modalbox)
- [返到顶部](http://www.seieiblog.top/backtop)
- [响应式导航栏](http://www.seieiblog.top/navbar)
- [菜单全选，反选](http://www.seieiblog.top/checkbox)
- [悬浮拉出菜单(可短时间横穿列表)](http://www.seieiblog.top/hoverpullout)
